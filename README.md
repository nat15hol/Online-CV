# CV Website

This is a simple personal CV built using HTML and CSS. The project is designed to present my professional profile, experience, education, and technical competencies in a clean and structured web format.

## Overview

The website is a static CV page that includes:

- Personal profile and summary
- Work experience (including internship at Region Skåne)
- Education history from Lund University and ongoing studies
- Technical competencies in data engineering, programming, and analytics
- Languages and additional information

## Built With

- HTML5
- CSS3

## Purpose

The purpose of this project is to:

- Practice building structured and responsive web layouts
- Present my CV in a modern digital format
- Showcase skills in HTML/CSS and data-related competencies
- Serve as a portfolio piece for future opportunities

Simply open `index.html` in any web browser.
